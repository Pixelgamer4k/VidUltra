LESSONS LEARNED
===============

1. GitHub Actions is viable for Android development without local IDE
2. Dependency management is critical in Android ecosystem
3. Resource files must match manifest exactly
4. Gradle Wrapper is essential for CI/CD
5. Version compatibility requires careful checking
6. Camera2 API needs proper thread management
7. Compose BOM simplifies but can conflict
8. Incremental fixes work better than large changes
